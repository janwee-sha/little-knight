# Little Knight

一个使用 Godot 4.7 制作的单场景横版动作 Demo。所有画面均由 GDScript 程序化绘制，不需要下载外部素材。

## 玩法

- `A / D` 或方向键：移动
- `Space / W / ↑`：跳跃（支持土狼时间与跳跃输入缓存）
- `J / Z`：挥剑，可击破敌人的火球
- `K / X / Shift`：短距离无敌闪避
- `R`：失败或通关后重新开始
- 手柄：左摇杆移动，A 跳跃，X 攻击，B 闪避

穿过五段废墟，躲开地刺和深坑，击败全部八名近战/远程守卫，然后抵达最右侧城门。地图中段的月光祭坛可恢复生命。

## 运行

在 Godot 中导入本目录的 `project.godot`，按 `F6`/`F5` 运行即可。
